# INT-332---DevOps
# DevOps Course Repository

This repository contains structured learning materials, detailed notes, and scenario-based task solutions for the DevOps course. It is maintained as part of continuous learning and weekly submissions.

---

## Repository Overview

The repository is organized to cover both theoretical concepts and practical understanding of DevOps tools and infrastructure. It includes:

* Conceptual notes for each unit
* Step-by-step explanations of core topics
* Scenario-based questions and answers
* Command examples and workflows

---

## Contents

### Unit 1: Basics of DevOps Infrastructure

Topics covered:

* Introduction to containers and evolution
* Container runtime and process isolation
* Namespaces and control groups (cgroups)
* Container images and layering
* Image registries and distribution
* Introduction to Docker
* Docker architecture (CLI, daemon, registry)
* Docker object types (containers, images, networks, volumes)
* Docker filesystem and layering

---

### Unit 2: Image Building and Container Management

Topics covered:

* Dockerfile concepts and instructions
* Image building process and layer caching
* Image tagging and versioning
* Image inspection (history and metadata)
* Docker networking (bridge, host, overlay)
* DNS and container communication
* Port mapping and exposure
* Docker storage (volumes and bind mounts)
* Copy-on-write mechanism
* Registries (Docker Hub, GHCR, private registries)
* Authentication and access tokens

---

## Scenario-Based Tasks

This repository also includes answers to practical and scenario-based questions such as:

* Clear explanation
